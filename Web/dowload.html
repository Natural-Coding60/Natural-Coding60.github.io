Package Managers

Install from the command line

Prefer to install from your package manager? We're in all the major ones.

npm
# Install globally npm install -g nexus-app # Verify installation nexus --version
pnpm / yarn
# pnpm pnpm add -g nexus-app # yarn yarn global add nexus-app
Homebrew (macOS / Linux)
# Add tap brew tap nexuscorp/tap # Install brew install nexus-app # Upgrade later brew upgrade nexus-app
Windows Package Manager (winget)
# Install winget install NexusCorp.Nexus # Upgrade later winget upgrade NexusCorp.Nexus
apt (Debian / Ubuntu)
# Add GPG key curl -fsSL https://pkg.nexus.dev/gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/nexus.gpg # Add repository echo "deb [signed-by=/etc/apt/keyrings/nexus.gpg] https://pkg.nexus.dev/apt stable main" | sudo tee /etc/apt/sources.list.d/nexus.list # Install sudo apt update && sudo apt install nexus-app
Snap
sudo snap install nexus-app --classic

Docker Image

Run Nexus in any CI/CD environment with our official Docker image. Available on Docker Hub and GitHub Container Registry.

Docker
# Pull latest docker pull nexuscorp/nexus:latest # Run a build docker run --rm -v $(pwd):/workspace nexuscorp/nexus:latest build
⚙️

GitHub Action

Official GitHub Action for Nexus. Set up the CLI in your workflow in one step.

.github/workflows/ci.yml
- uses: actions/setup-node@v4 - uses: nexuscorp/setup-nexus@v3 with: version: '3.0.4' - run: nexus build --affected

Security

Verify your download

All releases are signed and checksummed. Verify before installing.

Verify SHA-256
# Download checksums file curl -L https://releases.nexus.dev/v3.0.4/checksums.txt -o checksums.txt # Verify (Linux/macOS) sha256sum --check checksums.txt # Verify (Windows PowerShell) Get-FileHash Nexus-3.0.4-win-x64-setup.msi -Algorithm SHA256
